CodeVille

CodeVille is one of the biggest communities which teach people how to code in different programing
languages and introduce them to the world of web-design. This community is one of the most friendly
communities I've ever been in and if you want to learn to code I'd say this is one of the best choices
you can make. This is a discord comunity in which the teachers and the mentors teach the people that
enter everything they need to know about web-desing and programming.

Some things we study:

    • HTML - one of the most popular web-design language.
      CSS - this "language" describes how HTML elements are described on a web-page.
      JavaScript - another web-design language that is more complicated than HTML and is usually used for animations on a webpage.

    • In my opinion the teachers want to teach the sutdents the basics of programming and also teach those who really want to work in this domain all the stuff that they need before they begin their journey.
